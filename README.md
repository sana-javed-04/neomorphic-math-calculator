# 🧮 Neomorphic Glass-Style Web Calculator

A clean, modern, and highly interactive web-based mathematical calculator designed with fluid visual states and instant calculation evaluation engines. Built completely from scratch with custom layout overlays and computational logic.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic workspace layout blocks (`.cont`, input elements, semantic button divisions).
* **CSS3 Engine:** Explicit linear font styles, dark spatial background gradients (`#0a0a0a` to `#3a4452`), drop shadows, and glassmorphic backdrop filters (`blur(45px)`).
* **Vanilla JavaScript:** Multi-tier node conversion operations (`Array.from`), targeted arithmetic string interpolation mapping, slice controllers, and algebraic computation matrices (`eval`).

---

## ✨ Key Features Included

* **Instant Computation Parsing:** The evaluation layer maps mathematical operations dynamically using optimized validation try-catch blocks to catch calculations errors immediately (`Error`).
* **Micro-Interaction State Triggers:** Clean operator tracking nodes light up dynamic design tokens (`greenyellow`) for optimal screen contrast.
* **Smart Display Refresh Logic:** Sequential event listeners dynamically isolate old query result strings instantly upon new key interaction pipelines (`resultShow`).
* **Array-Based Node Allocation:** Uses strict array indexing methodologies (`forEach`) to bind interaction controllers across multi-tier grid button layouts seamlessly.
